### docker container

1. What is container?
<br> OS 커널을 여러 개의 격리된 공간으로 격리(Isolation)하여 독립적인 여러 개의 시스템처럼 사용하는 가상화
2. How to use docker
<br> (Windows) wsl 설치 후 도커 설치 -> 도커 컨테이너 실행
3. How to use git in docker container
<br> GitHub계정에 ssh키 저장 -> 저장소 만들기 -> 저장소 클론 -> 수정 및 Push
